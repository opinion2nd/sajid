[SIZE=6][B]🧠 How it works[/B][/SIZE]
Minecraft signs support [I]translation keys[/I]. The server shows the player a hidden sign containing a key like [ICODE]key.freecam.toggle[/ICODE]:
[LIST]
[*]A [B]vanilla[/B] client can't translate the unknown key, so it sends the [B]raw key[/B] back.
[*]A client with the [B]Freecam mod[/B] auto-replaces it with the mod's label ("Toggle Freecam").
[/LIST]
FreecamGuard reads the answer:
[LIST]
[*][B]Raw key[/B] returned → clean client → nothing happens.
[*][B]Translated[/B] text returned → mod detected → player kicked.
[*][B]Blank / blocked[/B] (client refused the probe) → treated as a dodge → kicked (strict mode).
[/LIST]
The fake sign is never placed in your world — the original block is restored instantly, and the sign re-pops until the client answers so it can't be ignored.

[SIZE=6][B]⚙️ Configuration[/B][/SIZE]
[CODE=yaml]
modDetection:
  probeOnJoin: true          # run the check when a player joins
  probeDelayTicks: 40        # delay before the sign pops (20 ticks = 1s)
  probeTimeoutTicks: 300     # how long to wait for an answer (300 = 15s)
  autoKick: true             # kick on detection
  kickOnNoResponse: true     # STRICT: kick clients that ignore/block the sign
  notifyAdmins: true         # alert freecamguard.notify staff
  kickMessage: "&c&l✖ Disallowed Mod Detected\n \n&7You are using &c&l{mod}&7.\n&7This is &cnot allowed&7 on this server.\n&7Remove it and rejoin to play."
  noResponseKickMessage: "&c&l✖ Anti-Cheat Check Failed\n \n&7You did not close the verification sign.\n&7Please &f&lrejoin&7 and &fclose the sign&7 that pops up."
  detect:
    Freecam: true
    Meteor Client: true
    Wurst: true

brandDetection:
  notifyAdmins: true         # alert on modded brand / cheat channel
  kickOnCheatChannel: true   # kick on a known cheat plugin channel
[/CODE]
Use [ICODE]&[/ICODE] colour codes and [ICODE]\n[/ICODE] for new lines. [ICODE]{mod}[/ICODE] = detected mod, [ICODE]{player}[/ICODE] = player name.

[SIZE=6][B]❓ FAQ[/B][/SIZE]
[B]Needs a client mod or resource pack?[/B] No — 100% server-side.
[B]Will it kick legit players?[/B] A normal player just closes the quick sign pop-up and is verified instantly. Only clients that block/ignore the check are kicked by strict mode (toggle it off if you prefer).
[B]Bedrock (Geyser/Floodgate)?[/B] Automatically skipped — the check is Java-only.
[B]Catches every cheat?[/B] It reliably catches mods that expose translatable keys (like Freecam) and dodgers via strict mode. For movement/combat cheats (fly, killaura, reach) pair it with a behavioural anti-cheat; for Freecam/ESP vision, pair it with a block-masking plugin.
